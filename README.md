
# ふぁいなるおぷしょん

合成音声を字幕と共にタイムラインへ投げ込むDaVinci Resolve用スクリプト  
安定動作とスクリプトのみでの完結をコンセプトに作成しています。代わりに低機能です。

最新版のダウンロード：
[Download](https://github.com/hasuno791/FinalOption/releases/latest)

## 使い方

1. ダウンロードした `finaloption.py` と `finaloptionフォルダ` をDaVinci Resolveのスクリプト用フォルダに配置します。<br><br>環境にもよりますが、おそらくこの辺にあります。<br>

   ```text
   C:\ProgramData\Blackmagic Design\DaVinci Resolve\Fusion\Scripts\Utility
   ```

2. `finaloptionフォルダ` 内の `config.json` を開き、使用環境に合わせて各項目を設定します。<br>
ファイル内に既に記入されているものは一例です。変更や削除を行っても動作に支障はありません。

| 項目 | 説明 |
|------|------|
| `audio_folder` | 投げ込みたい音声ファイルが保存されているフォルダを絶対パスで指定します。 |
| `audio_ext` | 投げ込みたい音声ファイルの拡張子を指定します。|
| `track_map` | キャラごとに音声ファイルと字幕を何番トラックに配置するかを指定します。<br>キャラ名については音声ファイルのファイル名、あるいは一緒に保存されているtxtファイル内に記載されているものと完全に一致する必要があります。名前の間のスペースの有無などはソフト毎に異なりますので注意してください。 |
| `replacements` | txtファイル内に字幕にしたくない文字列(プリセット名など)を削除、置換する場合はここで指定することができます。<br>一番先頭にある項目ではテキストの前後にある改行の削除を行っています。不要であれば削除してください。 |
| `max_line_length` | 字幕の自動改行をする文字数を指定します。 |
| `sleep_time` | DaVinci Resolveの内部処理を待つため、スクリプト側の処理を一時停止する時間を指定します。<br>動作が安定しない場合、この数値を増やすことで改善する可能性があります。 |

3. メディアプールのマスター直下に `text` と `voice` という名前のビンを作成します。

4. `text` ビンの中に、`config.json` 内の `track_map` に記載した名前と同名であるText+クリップを配置します。<br>

   Text+を作成するときは、作成する予定のプロジェクトファイルとfpsや解像度が一致しているかを確認してください。fpsが異なっている場合、音声と字幕が同じ長さにならない現象が発生します。<br>
   また、プロジェクトファイルのfpsが整数値ではない場合、正常に動作しない可能性があります。<br>

設定後は`config.json`で指定したフォルダにお好きなソフトウェアから音声ファイルをテキストファイルと共に出力し、スクリプトを実行して正常に動作することを確認してください。  
正常に動作しない場合、コンソールに表示されるログを確認すると原因がわかることがあります。

### Davinci Resolve初心者向けTips
- DaVinci Resolve内でスクリプトが確認できたら、スクリプトに対して任意のショートカットを割り当てることで起動を楽にできます。
- 作成した `text` ビンとその中身をパワービンに保存すると、プロジェクト変更後も使いまわしが可能です。
